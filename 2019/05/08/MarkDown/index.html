基本常用用法

加粗：**Strong**

_斜体_：* Emphasize*

—下划线—： ---下划线---or<u>下划线</u>

加粗斜体：***Strong & Emphasize***

删除线：~~删除线~~

分割线：***或---或___

选中标记：<mark>标记</mark>

转义字符\：So A\*maz\*ing So A*maz*ing

2的n+1次方：2<sup>n</sup> 2\^(n+1) = 2(n+1)

表格

主要使用：|列，-行，:对齐；

（PS：最左边和最右边|只是为了美观，: 作为左右对齐的标志；）

单表格

First Header Second Header
Content Cell Content Cell
Content Cell Content Cell

对齐+多表格

Left Aligned Center Aligned Right Aligned
col 3 is some wordy text $1600
col 2 is centered $12
zebra stripes are neat $1

复杂表格：

Option name Markup Result if enabled
Intra-word emphasis So A*maz*ing So Amazing
Strikethrough ~~Much wow~~ Much wow
Underline [^under] _So doge_ So doge
Quote [^quote] \”Such editor\” Such editor
Highlight \==So good\== So good
Superscript hoge\^(fuga) hogefuga
Autolink http://t.co http://t.co
Footnotes [\^4] and [\^4]: [^4] and footnote 4
