如何制作test.dmg镜像文件？

  • 新建文件夹test，将 *.app 和 Applications替身文件 放到这个文件夹下！

  • 打开磁盘工具 -> 文件 -> 新建镜像 -> 来自文件夹的镜像，选中文件夹test，存储即可生成！
